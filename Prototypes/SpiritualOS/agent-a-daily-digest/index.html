Design Notes — Contract Compliance
Content is matte, chrome is glass. Verse block, prayer list, and event cards all sit on #F8F4EC matte surfaces. The daily digest card itself is glass (blur 24px lensed). Glass-on-glass never occurs.
Formation chip is warm, never bold. "Reading · Day 14" is a small GlassChip in amenGold — not a headline number. No red/orange tones used on spiritual counters.
Berean suggestion is italic invitation. The saved Berean session row uses italic text in amenPurple — styled as an invitation, never a command or CTA button.
No engagement metrics. Timeline rows show relational context (who, what, when) but zero like/share/view counts are visible anywhere on this surface.
Top-edge shimmer on all glass elements. GlassCard, GlassChip, and collapsed chip all carry the warm white shimmer gradient at their top edge per the SharedComponents contract.
TimelineRow connector line. The vertical connector between timeline rows uses amenGold at 28–30% opacity. Completed rows show strikethrough with gold decoration; incomplete rows show color-coded status icons.
