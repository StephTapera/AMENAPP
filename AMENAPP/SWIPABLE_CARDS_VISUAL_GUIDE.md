## Performance Notes

- **Cards Rendered**: Maximum 3 at a time
- **Image Caching**: AsyncImage with placeholders
- **Gesture Recognition**: Native SwiftUI DragGesture
- **Animation**: Hardware-accelerated Spring animations
- **Memory**: O(1) - constant memory usage with card recycling
- **Network**: Lazy load suggested users in batches
